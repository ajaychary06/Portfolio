Fake News Detection Using Machine Learning

Built a Natural Language Processing pipeline to detect fake news from online articles using supervised learning techniques. The project combines text cleaning, feature extraction, and classification models to separate real and fake news effectively.

  • Preprocessed news articles with NLTK and regex
  • Extracted features using TF-IDF and CountVectorizer
  • Trained Logistic Regression, Random Forest, and SVM
  • Achieved over 92% accuracy with F1-score analysis
  • Evaluated using confusion matrix and precision-recall
